Day 1 - Introduction to Cybersecurity & Linux
### What is cybersecurity?
Cybersecurity is the practice of protecting systems, networks, and data from attacks or unauthorised access.

### Types of security
#### Offensive security
Offensive security is described as simulating a hacker's actions to find system vulnerabilities before real hackers do.

#### Defensive security 
Defensive security is the process of defending and securing devices and systems from attacks. Unlike offensive security, you do not attack systems, instead you monitoe and protect them.

### Cybersecurity roles
Security analyst 
Security engineer
Penetration Tester
Incident Responder

### Networking basics
#### IP addresses
IP addresses are used to identify a device on a network ( can be changed) 
#### MAC addresses
A MAC address is a unique identifier given to a device's network interface.

### Internet basics
The internet is a global network that connects devices and allow communications between them

### Linux commands practiced

#### pwd
Shows the current directory(shows where I am in the system)
#### ls
Lists files and folders between directories
#### cd 
Used to move between directories
#### mkdir
Creates a new folder
#### rm -r
Deletes a folder and everything in it
