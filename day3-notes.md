## Day 3 - Networking & Web Development Basics

## How the Internet Works
- The internet is a global network connecting devices
- Communication happens between:
   - Client (your device)
   - Server (website/server)
- When visiting a website:
   - The browser sends a request
   - The server responds with data (webpage)

## Ports
- Ports are entry points on a computer used by different services
- Each service runs on a specific port
Examples:
  - Port 80 → HTTP
  - Port 443 → HTTPS
  - Port 22 → SSH
## TCP vs UDP
  ### TCP (Transmission Control Protocol)
  - Reliable
  - Ensures data is delivered correctly and in order
  - Slower
  ### UDP (User Datagram Protocol)
   - Faster
   - Does not guarantee delivery
   - Used in streaming, gaming, etc.

## DNS (Domain Name System)
 - Converts domain names into IP addresses
Example:
 - google.com → IP addresses
  ### How DNS Works:
  1. User enters a website name
  2. DNS server finds the IP address
  3. Device connects to the server using that IP
